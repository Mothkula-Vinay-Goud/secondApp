## üöÄ Features
The app consists of a 3-screen onboarding sequence managed by a `UINavigationController`:

1.  **User Information Screen**: Features custom-styled `UITextField` components for capturing the user's Name and Username.
2.  **Profile Image Selection**: A dedicated screen for photo uploads featuring a circular UI overlay (Note: Image picker integration is currently in development).
3.  **Notifications Screen**: A final call-to-action screen to request push notification permissions from the user.
